    AEGEAN






    Αποστολή μας


    Να ταξιδεύουμε με ασφάλεια τους επιβάτες μας, προσφέροντας πάντα υψηλό επίπεδο υπηρεσιών σε όλα τα στάδια του ταξιδιού, με ένα εκτενές δίκτυο δρομολογίων που συνδέει κάθε γωνιά της χώρας μας, αλλά και την Ελλάδα με το εξωτερικό.


























    VODAFONE






    Αποστολή μας


    Δρώντας ως ένας καταλύτης για την ψηφιακή κοινωνία του αύριο, η Vodafone δεσμεύεται να χρησιμοποιήσει τα δίκτυα και τις τεχνολογίες της ώστε να συνδέσει τους ανθρώπους για ένα καλύτερο μέλλον, που θα προσφέρει περισσότερες ευκαιρίες σε όλους. Είμαστε αποφασισμένοι να αξιοποιήσουμε τις τεχνολογίες μας για έναν ανώτερο σκοπό για να συμβάλλουμε στην πρόοδο, να διασφαλίσουμε την ισότιμη συμμετοχή όλων σε αυτήν την ψηφιακή κοινωνία και να αναλάβουμε δράση για την προστασία του πλανήτη.


























    PHILIPS









    Αποστολή μας


    Η αποστολή της Philips είναι να βελτιώσει τη ζωή των ανθρώπων μέσω ουσιαστικής καινοτομίας. Για να επιτευχθεί αυτό απαιτείται μια συνδεδεμένη επιχείρηση που επιτρέπει στις καλύτερες ιδέες της να φτάνουν γρήγορα στην αγορά.
























    COCA-COLA









    Αποστολή μας




    Η δήλωση αποστολής της Coca Cola είναι «να ανανεώσει τον κόσμο στο μυαλό, το σώμα και το πνεύμα, να εμπνεύσει στιγμές αισιοδοξίας και ευτυχίας μέσω των επωνυμιών και των ενεργειών μας και να δημιουργήσει αξία και να κάνει τη διαφορά». Η Coca Cola είναι μια εταιρεία που εστιάζει στο να αφήσει μια κληρονομιά όπου κι αν δραστηριοποιείται.



























    MICROSOFT









    Αποστολή μας




    Η δήλωση αποστολής της Microsoft είναι «να ενδυναμώσει κάθε άτομο και κάθε οργανισμό στον πλανήτη να επιτύχει περισσότερα». Η «ενδυνάμωση» είναι ο βασικός όρος σε αυτή τη δήλωση αποστολής.




























    MASTERCARD









    Αποστολή μας




    Η Mastercard είναι μια παγκόσμια εταιρεία τεχνολογίας στον κλάδο των πληρωμών. Η αποστολή τους είναι να συνδέσουν και να τροφοδοτήσουν μια χωρίς αποκλεισμούς, ψηφιακή οικονομία που ωφελεί όλους, παντού, κάνοντας τις συναλλαγές ασφαλείς, απλές, έξυπνες και προσβάσιμες.
